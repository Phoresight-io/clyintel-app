# Types

This folder contains shared TypeScript types for the ClyIntel product.

## Contents

- `supabase.ts` — generated Supabase types (run `supabase gen types` to regenerate)
- Other shared types as needed

## Rules

- Do not manually edit `supabase.ts`. Regenerate it using the Supabase MCP or CLI.
- Keep types as narrow and specific as possible.
